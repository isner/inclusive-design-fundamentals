Forms

Forms should give users the information they need to fill in the fields correctly, and should prevent user error as much as possible.

Users Affected

Blind

Low Vision

Mobility

Cognitive

Checklist

  • Labels for fields and groups of fields are visible at all times.

  • Labels accurately describes what each field is for.

  • It is obvious which fields in the form are required and which are optional.

  • Help text is provided for fields that require a specific data format.

  • Error messages clearly instruct the user on how to correct the error.

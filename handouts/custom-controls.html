Custom Controls

Use custom controls only when they are truly needed. Specify names, states, and values for every part of the control, as well as all mouse/keyboard interactions.

Users Affected

Blind

Low Vision

Mobility

Checklist

  • Give custom controls accessible names

  • Convey the state of custom controls that are capable of changing state

  • Define all keyboard interactions for custom controls

  • Limit the functionality of custom controls - do one thing well
